自然小调

C大调 C D E F G A B C

c小调 C D Eb F G Ab Bb C

​ 全 半 全 全 半 全 全

a小调 A B C D E F G A

关系大小调

a小调是C大调的关系小调；C大调是a小调的关系大调。

之前学习的五度循环圈，外围的12个调，内部的三个等音调，这15个大调都有对应的关系小调。大调主音下方小三度的，就是关系小调，如下图所示。

音程的度数判断

基本音级

C D E F G A B

判断

写出音程

比如A-B，C-E，F#-G

看两音之间包含的基本音级数

→确定几度

而不是看两者间的半音数量。

先确定度数→再看半音数→确定具体音程

自然音程与变化音程

自然音程

大调（或自然小调）里，任何两个音构成的音程都属于自然音程。

上图中，根音是位于下方的音，冠音是位于上方的音。

自然音程一共有4大类，14种：

  1. 纯音程：纯一度，纯四度，纯五度，纯八度；
  2. 大音程：大二度，大三度，大六度，大七度；
  3. 小音程：小二度，小三度，小六度，小七度；
  4. 三全音：增四度，减五度。

变化音程

自然音程之外的都是变化音程。

变化音程也有4大类：

  1. 增音程（除增四度之外）；
  2. 减音程（除减五度之外）；
  3. 倍增音程；
  4. 倍减音程。

增音程：以纯音程和大音程为基础，度数不变，增加一个半音，就是增音程。例：

​ F - G# 增二度 F - G 大二度

减音程：以纯音程和小音程为基础，度数不变，减少一个半音，就是减音程。例：

​ G# - Ab 减二度 G# - A 小二度

倍增音程：以增音程为基础，度数不变，增加一个半音，就是倍增音程。例：

​ F - Gx 倍增二度 F - G 大二度 F - G# 增二度

倍减音程：以减音程为基础，度数不变，减小一个半音，就是倍减音程。例：

​ G# - Abb 倍减二度 G# - A 小二度 G# - Ab 减二度

注意：纯一度没有减音程和倍减音程。

小调的变体

小调可以变成和声小调和旋律小调

Ⅰ Ⅱ Ⅲ Ⅳ Ⅴ Ⅵ Ⅶ Ⅰ
a自然小调 A B C D E F G A
a和声小调 A B C D E F G# A
a旋律小调 A B C D E F# G# A

和声小调中，将自然小调的第七音级升高半音，包含一个增二度。

旋律小调中，将自然小调的第六、第七音级升高半音。

旋律小调的上下行旋律不同，下行的时候还原。

三种小调和大调的关系

Ⅰ Ⅱ Ⅲ Ⅳ Ⅴ Ⅵ Ⅶ Ⅰ
C大调 C D E F G A B C
c自然小调 C D Eb F G Ab Bb C
c和声小调 C D Eb F G Ab B C
c旋律小调(上行) C D Eb F G A B C

Slurm集群管理教程

在进行管理之前，必须知道Slurm调度系统分为两个基本层面：Account（账户）和User（用户）。基于这两个基本层面，Slurm形成了“关联”(Assoc)。

Account类似于group的概念，是由多个User（用户）组成，在大集群上可以用来统一计算用户组的机时，通过对Account设置，可以对该用户组的全部用户进行条件约束。例如：在集群上需要限制某个用户组的使用总核数，则可以将这些用户放置在一个Account下，设置其可用的CPU总数、GRES等资源的限制。

User则落实到具体的某个用户，除了可以在Account上对全体用户进行限制外，还可以单独限制某个用户的最大资源。

1. 查看关联关系

1.1 查看全部关联关系

2.1 新建一个account的命令如下：
# sacctmgr add account [账号名称]
2.2 添加用户到指定的Account（例如tensorflow）：
# sacctmgr add user account=tensorflow
2.3 修改用户属性
# sacctmgr modify user [用户名] set [属性]=[设定值]

用户的属性值可以查阅帮助文档。

3. Account和User的权限管理

Account和User的权限使用GrpTRES来进行限制：

GrpTRES=<TRES=max TRES,…>
Maximum number of TRES running jobs are able to be allocated in aggregate for this association and all associations which are children of this association. To clear a previously set value use the modify command with a new value of -1 for each TRES id.
NOTE: This limit only applies fully when using the Select Consumable Resource plugin.

例如，限制lily用户的CPU核数为40，节点数为2，GPU最大使用为4：

# sacctmgr modify user lily set Grptres="cpu=40,node=2,gres/gpu=4"
# slassoc
   Cluster    Account       User                  QOS  Partition                        GrpTRES
---------- ---------- ---------- -------------------- ---------- ------------------------------
  abhpc-ai       root                          normal
  abhpc-ai       root       root               normal
  abhpc-ai tensorflow                          normal
  abhpc-ai tensorflow      abhpc               normal
  abhpc-ai tensorflow       lily               normal                  cpu=40,gres/gpu=4,node=2

重置该用户全部限制则为：

# sacctmgr modify user lily set Grptres="cpu=-1,node=-1,gres/gpu=-1"
   Cluster    Account       User                  QOS  Partition                        GrpTRES
---------- ---------- ---------- -------------------- ---------- ------------------------------
  abhpc-ai       root                          normal
  abhpc-ai       root       root               normal
  abhpc-ai tensorflow                          normal
  abhpc-ai tensorflow      abhpc               normal
  abhpc-ai tensorflow       lily               normal

同样，可以对整个Account进行限制，其含义是Account包含的全部用户累计总和不得超过最大限制。例如：

# sacctmgr modify account tensorflow set Grptres="cpu=40,node=2,gres/gpu=4"
# slassoc
   Cluster    Account       User                  QOS  Partition                        GrpTRES
---------- ---------- ---------- -------------------- ---------- ------------------------------
  abhpc-ai       root                          normal                                           
  abhpc-ai       root       root               normal                                           
  abhpc-ai tensorflow                          normal                  cpu=40,gres/gpu=4,node=2
  abhpc-ai tensorflow      abhpc               normal                                           
  abhpc-ai tensorflow       lily               normal

重置该Account的全部限制为：

# sacctmgr modify account tensorflow set Grptres="cpu=-1,node=-1,gres/gpu=-1"

4. 管理员计费系统

如果集群涉及到计费问题，则需要统计用户(User)或账户(Account)在某段时间内的使用机时，在用户教程中已经提供了用户机时的统计方法，但同一Account下的其他用户的作业是互相不可见的，因此，作为管理员可以对单个用户计费，也可以对整个Account进行机时计费，以下举例说明。

4.1 对用户lily自2019年1月1日0时起使用的机时进行统计：
# sacct -u lily -S 2019-01-01T00:00:00 -o "jobid,partition,account,user,alloccpus,cputimeraw,state,workdir%60" -X |awk 'BEGIN{total=0}{total+=$6}END{print total}'
4.2 对名为tensorflow的account自2019年1月1日0时起使用的机时进行统计：
# sacct -A tensorflow -S 2019-01-01T00:00:00 -o "jobid,partition,account,user,alloccpus,cputimeraw,state,workdir%60" -X |awk 'BEGIN{total=0}{total+=$6}END{print total}'

注意这里的机时单位都是秒，换算成核时需要除以3600。

音乐学习

乐理（一）

基本概念

拍：节拍单位，用固定的音符来代表。
速度：每分钟的拍数。（bpm=beats per minute）
小节：一定数量的拍构成。
节拍：（4/4是最常用的节拍，上面的4代表每小节为4拍，下面的4代表以4分音符长度为一拍）以此类推。
时值：音乐时间值的表示法。比如：全音符，二分音符，四分音符…

大调

每一个大调为七个相差“全全半全全全半”的音组成。
以C大调为例：CDEFGAB,CD键之间相差全音，DE键之间相差全音，EF键之间相差半音，以此类推至BC键之间相差半音。
D大调：D E F# G A B C# D
F大调：F G A Bb C D E F
在一个大调里音的稳定性排序（从稳定到不稳定）：1 5 3 6 2 4 7（前三个为稳定音，后三个为不稳定音）。
音的倾向性：从不稳定音运动到稳定音，比如

$$1 \longleftarrow 2\qquad 3 \longleftarrow 4\qquad 5 \longleftarrow 6\qquad 7 \longrightarrow 1$$

音程

音程：表示两个音之间的高低关系，可以用包含的半音数量来区分。

音程转换规律：转为前后相加为九度；大小互换；增减互换；纯音程转位后仍是纯音程。（例如大七度转位后为小二度）

音程的协和度

同时出现两个音的协和程度。
完全协和音程 纯一度，纯八度
协和音程 纯四度 纯五度
不完全协和音程 大小三度 大小六度
不协和音程 大小二度 大小七度 增四减五度

和弦

和弦是一些音的结合，一般按照三度叠置来结合的。

并不是所有的和弦都按照三度叠置。

大三和弦

Cmaj （C） C E G （大三度+小三度）

C 根音

E 三音

G 五音

CE音程为大三度，EG音程为小三度，三音与根音之间的关系为三度，五音与根音之间的关系为五度。

小三和弦

Dmin（Dm）D F A（小三度+大三度）

C大调和弦

1、4、5这三个音级上构成的是大三和弦；

2、3、6这三个音级上构成的是小三和弦。

那么7构成的就是减三和弦。

减三和弦

Bdim ($B^\circ$) B D F （小三度+小三度）

大七和弦（大大七和弦）

Cmaj7（CM7） C E G B（大三度+小三度+大三度）

相当于大三和弦再加一个距离根音大七度的音。

小七和弦

Amin （Am7） A C E G （小三度+大三度+小三度）

相当于小三和弦再加一个距离根音小七度的音。

属七和弦

Gdom7 (G7) G B D F (大三度+小三度+小三度)

相当于大三和弦再加一个距离根音小七度的音。

小大七和弦

$Emin^{maj7}$ (Em^{M7}) E G B D# (小三度+大三度+大三度)

相当于小三和弦再加一个距离根音大七度的音。

和弦转位

以C为例：
原位（根音在最下）：CEG——C
第一转位（三音在最下）：EGC——C/E
第二转位（五音在最下）：GCE——C/G
七和弦的转位以此类推。

七和弦转位

原位 CEGB——CM7

第一转位 EGBC——CM7/E

第二转位 GBCE——CM7/G

第三转位 BCEG——CM7/B

五度循环圈

一个音与上方纯五度的音是最密切的，两个纯五度的音同时响仅次于纯一度和纯八度的协和。

调式中各音级的名称

CDEFGAB分别为：主音，上主音，中音，下属音，属音，下中音，导音。

Do Re Mi Fa Sol La Si
1 2 3 4 5 6 7
C D E F G A B
